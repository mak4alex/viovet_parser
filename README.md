# Viovet Parser

Extracts products from [www.viovet.co.uk](https://www.viovet.co.uk/ "Pet shop") category page and loads to CSV file.

## Usage

### Example

See main.rb file for example

### Run Example

1. Clone or download repo
2. $ cd viovet_parser
3. $ gem install bundle
4. $ bundle install
5. $ ruby main.rb

## RSpec Tests

Run tests:
  $ rspec spec
